# Backend Engineer Playbook Index

## 목적

백엔드 엔지니어링 지식을 단순 암기가 아니라 실무 문제 해결, 설계 판단, 면접 설명, AI 활용까지 연결해 체계화한다.

## 핵심 흐름

1. 개념을 이해한다.
2. 동작 원리를 설명한다.
3. 실무 문제와 연결한다.
4. 해결 패턴과 한계를 정리한다.
5. 사례로 검증한다.
6. 면접 답변으로 압축한다.
7. AI를 활용해 반복 개선한다.
