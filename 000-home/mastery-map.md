# Mastery Map

## 숙련도 기준

| 단계 | 기준 |
|---|---|
| L0 | 들어본 적만 있음 |
| L1 | 한 줄 정의 가능 |
| L2 | 동작 원리 설명 가능 |
| L3 | 실무 문제와 해결책 설명 가능 |
| L4 | 프로젝트 경험과 연결해 면접 답변 가능 |
| L5 | 다른 사람에게 가르칠 수 있음 |

## 운영 방법

- 현재 레벨은 솔직하게 적는다.
- 매주 한 번만 업데이트한다.
- L0에서 L5로 한 번에 올리려 하지 않는다.
- L3 이상부터는 반드시 실무 사례나 예제 코드를 연결한다.
